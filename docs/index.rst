Welcome to photomacros's documentation!
========================================


:mod:`photomacros` is a **machine learning** project built with PyTorch. It creates a model to determine the macronutrient breakdown of a meal based on a photo. The project utilizes the `Food101 <https://www.kaggle.com/datasets/dansbecker/food-101/>`_, a large dataset of of 101 food categories in 101,000 images. 



:


.. hlist::
   :columns: 1

   * Word segmentation (tokenization)
   * Lemmatization
   * POS tagging
   * Morphological features analysis
   * Dependency Parsing
